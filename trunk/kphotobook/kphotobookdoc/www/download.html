Download KPhotoBook

Downloads and cvs access are available on the projectsite at sourceforce.net: KPhotoBook at sourceforge.net

Building from CVS

Follow these instructions to build the application from the repository:

  • Login to the cvs-repository at sourceforge (no password is required): 
    cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/kphotobook login
  • Checkout the sourcecode: 
    cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/kphotobook co kphotobooksrc
  • Change into the checked out directory
    cd kphotobooksrc
  • Create the makefile: 
    make -f Makefile.cvs
  • Start configure (where KDE-installation-path is '/opt/kde3' for example): 
    ./configure --prefix='KDE-installation-path'
  • Build the application: 
    make
  • Change to root: 
    su
  • Install the application: 
    make install

Building the tarball

Follow these instructions to build the application from the tarball:

  • Downloaded the tarball from KPhotoBook at sourceforge.net.
  • Extract the downloaded tarball: 
    tar -xzf kphotobook-x.x.x.tar.gz
  • Go into the extracted directory: 
    cd kphotobook-x.x.x
  • Start configure (where KDE-installation-path is '/opt/kde3' for example): 
    ./configure --prefix='KDE-installation-path'
  • Build the application: 
    make
  • Change to root: 
    su
  • Install the application: 
    make install
