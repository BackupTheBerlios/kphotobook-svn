Building from CVS

Follow these instructions to build the application from the repository:

  • Login to the cvs-repository at sourceforge: cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/kphotobook login
  • Checkout the sourcecode: cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/kphotobook co kphotobooksrc
  • Create the makefile: make -f Makefile.cvs
  • Start configure: ./configure --prefix='KDE-installation-path' (where KDE-installation-path is '/opt/kde3' for example)
  • Build the application: make
  • Change to root: su
  • Install the application: make install
