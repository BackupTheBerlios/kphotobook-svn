Handle Tags Recursively When setting filters on a parent tags, files associated with the sub tags must be considered too. E.g. if dog is a sub tag of animal then, when searching for images containing an animal, the images tagged with dog must be displayed too. Mike March 2004 0.0.5
Tag Tree The tag tree is one of the main elements wich makes the work with KPhotoBook easy. The tag tree allows you to create your own tags which are also sorted hierarchical. The tags can be labelled with a text and an individual icon. Mike March 2004 0.0.4
Show Photos The images themselves are shown in a separate frame of the program Konqueror-like. It is possible to change the size of the thumbnails. Mike February 2004 0.0.3
Directory Tree The directory tree is one of the main elements of the userfriendly interface. It allows you to add and remove folders containting images for the current database. The folders can be added single or recursively. The directory tree also allows you to filter the images so that only the pictures of the chosen directories are shown. Mike February 2004 0.0.3
