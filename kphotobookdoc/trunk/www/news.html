2004.06.20

Mike		KPhotoBook
There is a second official release of KPhotoBook tagged as version 0.0.6. There is not a lot new stuff (hey guys, it's summer in Switzerland... ;-)):
  • Autoconf and automake files updated. (Should work with newest versions now.)
  • There are menu entries for showing/restoring all toolviews (Tagtree, Sourcedirectories, EXIF).
  • Tagging of images can be locked/unlocked to avoid accidental change of tags.
  • Filters and tree states (open closed nodes) are remebered on a per file and per user basis.
  • A mimetype for KPhotoBook associating '*.kpb' files with KPhotoBook is installed.
  • Symbolic links to the matched or selected images can be created in a chosable directory.
  • Several minor changes and improvements...
Please try this new version and have fun with it.
