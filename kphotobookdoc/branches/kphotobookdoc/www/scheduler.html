Directory TreeThe directory tree is one of the main elements of the userfriendly interface. It allows you to add and remove folders containting photos for the current database. The folders can be added single or recursively. The directory tree also allows you to filter the photos so that only the pictures of the chosen directory are shown.February 2004
Show PhotosThe photos themselves are shown in a separate frame of the program. The photos are shown konqueror-like. It is possible to change the size of the thumbnails with a ruler. February 2004

FeaturesDescriptionPlannedRelease
Tag TreeThe tag tree is one of the main elements wich makes the work with KPhotoBook easy. The tag tree allows you to create your own tags which are also sorted hierarchical. There are several types of tags which allows you to type in different types of data. The tags can be labelled with a text and an individual icon. March 2004
